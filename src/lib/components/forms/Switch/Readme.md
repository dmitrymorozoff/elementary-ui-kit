### Basic Usage:

Switch is used for switching between two opposing states.

```jsx
<div style={{ margin: "8px" }}>
    <Switch />
</div>
<div style={{ margin: "8px" }}>
    <Switch checked={true} />
</div>
```

### Disabled

```jsx
<div style={{ margin: "8px" }}>
    <Switch disabled={true}  />
</div>
<div style={{ margin: "8px" }}>
    <Switch disabled={true} checked={true}  />
</div>
```

### Default checked

```jsx
<div style={{ margin: "8px" }}>
    <Switch defaultChecked={true} />
</div>
```
